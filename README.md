# 📊 Student Performance Analysis

## 📌 Project Description
This project analyzes student marks using Python to understand overall academic performance. It calculates key metrics like average marks, highest and lowest scores, and pass/fail statistics.

---

## 🎯 Objectives
- Calculate average marks of students  
- Identify highest and lowest marks  
- Determine pass and fail count  
- Visualize results using graphs  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Google Colab  

---

## 📁 Dataset
The dataset is created using Python and contains student names and their marks.

---

## 📊 Features
- Average marks calculation  
- Highest & lowest marks detection  
- Pass/Fail classification  
- Graphical representation (bar chart)  

---

## ▶️ How to Run
1. Open the notebook in Google Colab  
2. Run all cells  
3. View results and graph  

---

## 📈 Output
The project generates:
- Average marks  
- Highest marks  
- Lowest marks  
- Pass/Fail count  
- Visualization chart  

---

## 🚀 Conclusion
This project helps in analyzing student performance efficiently and provides useful insights for academic improvement and decision-making.
